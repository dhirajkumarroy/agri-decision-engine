About Farmpilot AI

Empowering farmers with intelligent crop recommendations using Machine Learning & Live Weather Data.

Project Objective

Farmpilot AI is designed to assist farmers and agricultural researchers in making informed decisions about the best crop to cultivate. It blends soil nutrient analysis, rainfall patterns, and real-time weather updates with machine learning predictions — delivering results in seconds.

Key Features

ML Crop Prediction

Our model analyzes soil NPK values, pH, rainfall & weather to compute the most suitable crops.

Live Weather Integration

Real-time temperature, humidity & climate data enhances recommendation accuracy.

High Accuracy

Model outputs top 5 recommended crops with confidence scores for easy decision-making.

How It Works

Step 1 — Input Soil & Weather Data

User provides NPK, pH, rainfall, and location for live weather.

Step 2 — Weather Fetched Automatically

System retrieves real-time weather data for the selected city.

Step 3 — ML Model Predicts Crops

The model computes suitability and returns top crop recommendations.

Step 4 — Results with Probability

The interface displays crops with images and match percentages.

Technology Stack

Project Details

  • ✔️ Supports 22+ crops with accurate ML predictions
  • ✔️ Uses FastAPI backend with Jinja templates
  • ✔️ Powered by live weather API (temperature, humidity)
  • ✔️ Clean UI with Bootstrap 5 & animations
  • ✔️ Easy to extend: can integrate fertilizer suggestions, crop calendar, etc.
